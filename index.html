About

  • The Web site is create for sending students test report to the parents.

Guidlines

  • First we have to create shareable google excel sheet with our college mail ID in Google. click to refer create google cloud excel file .
  • File name should be in "Year-studying-department" form. click to refer Filename.
  • Sheet name in excel file should be Examination field in form line 'IAT-1'. click to refer sheet name.
  • Fill students marks and other detials in excel file with required rows and columns. click to refer Format.
